Server:

A Servereken a SERVICES fülön lehet állítani.

-WEB

Be kell kapcsolni és lehet változtatni a tartalmát a weboldalnak.
Vagy URL vagy IP címmel lehet csatlakozni a PC "Web browser" alkalmazásával.

-DNS

Meg kell adni a Name: {URL} és egy Address: {IP cím}, valamint bekapcsolni.

-FTP

Az FTP serveren meg kell adni a felhasználó nevét, jelszavát és jogosultságait.
Egy gép consolejából kell az "ftp {URL/IP}" paranccsal csatlakozni majd bejelentkezni.

A parancsok:

  • dir: listázás
  • put: feltöltés
  • get: letöltés
  • delete: törlés
  • quit: kilépés
  • rename: átnevezés

-E-mail

Meg kell adni a Domain nevét.
Ez után hozzá lehet adni felhasználóneveket jelszóval.
A PC-n az "Email" alkalmazásban a "Configure mail" opciónál be kell állítani a

  • Your name: {neved}
  • Email address: {valami}@{domain név}
  • Incoming Mail Server: {URL/IP}
  • Outgoing Mail Server: {URL/IP}
  • User name: {felhasználónév}
  • Password: {jelszó}

Ha kész akkor a PC "Email" alkalmazásába üzenetet lehet küldeni a {másik felhasználó}@{domain név} számára.

-DHCP

A hálózat gépeihez automatikusan ad IP címet.

  • Pool name: {dhcp neve}
  • Default Gateway: {router IP cím}
  • DNS Server: {dns IP címe}
  • Start IP Address: {IP cím}
  • Subnet Mask: {Maszk}
  • Maximum Number of Users: {Max. felhasználók száma}
