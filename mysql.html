Jelzések

'name'ㅤ-ㅤAdott dolognak a neve

'condition'ㅤ-ㅤAdott dolognak megadottt feltétel

typeㅤ-ㅤAdott dolog típusa ami lehet:

  • INTㅤ-ㅤEgész számok halmaza
  • DECIMALㅤ-ㅤValós számok halmaza
  • VARCHARㅤ-ㅤBetűkarakter
  • DATEㅤ-ㅤDátum
  • TIMEㅤ-ㅤIdő
  • DATETIMEㅤ-ㅤDátum és idő
  • 'type'(20)ㅤ-ㅤA zárójelben a szám a maximum megadható karakterszám

functionㅤ-ㅤAdott dolog funkciója ami lehet:

  • CURRENT_DATE()ㅤ-ㅤA pontos jelenlegi dátumot adja meg
  • CURRENT_TIME()ㅤ-ㅤA pontos jelenlegi időt adja meg
  • NOW()ㅤ-ㅤA pontos jelenlegi dátumot és időt adja meg

constraintㅤ-ㅤAdott dolog elvárása ami lehet:

  • UNIQUEㅤ-ㅤNem ismétlődhet az oszlopban adat
  • NOT NULLㅤ-ㅤAz oszlop értéke nem lehet NULL

Alapok

sudo mysqlㅤ-ㅤMySQL elindítása

CREATE DATABASE 'name';ㅤ-ㅤLétrehoz egy adatbázist a megadott néven

DROP DATABASE 'name';ㅤ-ㅤKitörli a megadott adatbázist

USE 'name';ㅤ-ㅤA megadott nevű adatbázist fogja használni

show databases;ㅤ-ㅤMegmutatja az összes adatbázist

show tables;ㅤ-ㅤMegmutatja az összes táblát

DESCRIBE 'name';ㅤ-ㅤMegmutatja a táblázat felépítését

SELECT 'name' FROM 'name';ㅤ-ㅤA megadott oszlopokat írja ki a kiválasztott táblázatban

SELECT * FROM 'name';ㅤ-ㅤAz összes oszlopot kiírja a kiválasztott táblázatban

SELECT * FROM 'name' WHERE 'condition';ㅤ-ㅤAz összes oszlopot kiírja a kiválasztott táblázatban a feltételnek megfelelően

CREATE TABLE 'name' (ㅤ-ㅤTáblázat létrehozása

'name' type,ㅤ-ㅤHozzáad egy oszlopot a megadott névvel és értékkel rendelkezve

'name' type UNIQUEㅤ-ㅤHozzáad egy oszlopot a megadott névvel és értékkel rendelkezve amiben nem ismétlődhet az érték

);

Formázások

ALTER DATABASE 'name' READ ONLY = 1;ㅤ-ㅤCsak olvasni lehet az adatbázist

ALTER DATABASE 'name' READ ONLY = 0;ㅤ-ㅤÍrni és olvasni is lehet az adatbázist

ALTER TABLE 'name' RENAME COLUMN 'name' TO 'name';ㅤ-ㅤA kiválasztott táblázat megadott oszlopának régi nevét az újra váltja

ALTER TABLE 'name' MODIFY COLUMN 'name' type;ㅤ-ㅤA kiválasztott táblázat megadott oszlopának az értékét váltja a kiválasztottra

ALTER TABLE 'name' MODIFY 'name' type FIRST;ㅤ-ㅤA kiválasztott táblázat megadott oszlopát helyezi az első helyre

ALTER TABLE 'name' MODIFY 'name' type AFTER 'name';ㅤ-ㅤA kiválasztott táblázat megadott oszlopát helyezi a kiválasztott oszlop mögé

ALTER TABLE 'name' MODIFY 'name' type constraint;ㅤ-ㅤA kiválasztott táblázat

ALTER TABLE 'name' DROP COLUMN 'name';ㅤ-ㅤA kiválasztott táblázat megadott oszlopát kitörli

ALTER TABLE 'name' ADD CONSTRAINT constraint('name');ㅤ-ㅤA kiválasztott táblázat megadott oszlopának az elvárását állítja

ALTER TABLE 'name' ADD CONSTRAINT 'name' CHECK('condition')ㅤ-ㅤHozzáad egy ellenőrzést a feltételnek megfelelően

UPDATE 'name' SET 'condition';ㅤ-ㅤA kiválasztott táblázat megadott feltétele alapján átírja az oszlop összes adatát

UPDATE 'name' SET 'condition' WHERE 'condition';ㅤ-ㅤA kiválasztott táblázat megadott feltétele alapján átírja az oszlop adatát a beírt feltételnek megfelelő sorban
