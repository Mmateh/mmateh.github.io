Attribútumok

Fájlok és könyvtárak tulajdonsáhai

Jogosultsági szintek

r - olvasási jogosultság (read)

w - írás jogosultság (write)

x - végrehajtási jogosultság (execute)

- - nincs jogosultság

Jogosultsági szintek numerikusan

r (read) - Az értéke: 4

w (write) - Az értéke: 2

x (execute) - Az értéke: 1


7 - olvasás, írás és végrehajtás

6 - olvasás és írás

5 - olvasás és végrehajtás

4 - olvasás

3 - írás és végrehajtás

2 - írás

1 - végrehajtás

0 - nincs jogosultság

Tulajdonlás

u - tulaj (user)

g - csoport (group)

o - egyéb (others)

Fájltípusok

- - fájl (file)

d - könyvtár (directory)

l - szimbolikus link (symlink)

b - blokk (block)

c - karakter (character)

p - cső (pipe)

s - csatlakozó (socket IP+port)

Jogosultságok

ls -laF - Minden fájl és mappa jogosultságai kiírása

ls -laF fájl - A megadott fájl jogosultságai kiírása

sudo chmod u+jogosultsági_szint,g+jogosultsági_szint,o+jogosultsági_szint fájl/mappa - Fájlok és mappák jogosultságainak módosítása

sudo chmod tulajdonos=jogosultsági_szint fájl/mappa - A megadott tulajdonos számára engedélyezi a megadott jogosultságokat a fájlhoz vagy mappához

sudo chmod jogosultságok_numerikusan - Számok alapján adja meg a jogosultságot ugo sorrendben

sudo chown -R felhasználó mappa - A mappa tulajdonosának rekurzív módosítása a benne lévő tartalommal együtt

sudo chgrp csoportnév fájl - A fájl csoportjának megváltoztatása

Meghajtók

lsblk - Kilistázza a meghajtókat

sudo fdisk /dev/meghajtó - Meghajtó partícionálása

  • n - Új partíció létrehozása
    1. p - Elsődleges partíció
    2. default - Alapértelmezett partíciószám
    3. default - Első szektor értéke
    4. default - Utolsó szektor értéke
  • t - Partíció típus kiválasztása
    1. 83 - Linux (RAID-hez ez kell)
    2. 00 - Empty
    3. 0b - FAT32
    4. 82 - SWAP
  • w - Elmenti a beállításokat

sudo mkfs.ext4 -L elnevezés /dev/meghajtó - Elnevezi a meghjtót és ext4 fájlrendszert tesz rá

df -h - A fájlrendszer lemezhasználatának kiiratása olvasható módon


dd if=/dev/1.meghajtó1 of=/dev/2.meghajtó1 - Alacsony szintű másolás az első meghajtóról a második meghajtóra

Ubuntu Server

RAID1

  1. Meghajtók formázása
  2. sudo mkdir /mnt/mappanév - Csatolási könyvtár létrehozása
  3. sudo mount /dev/meghajtó1 /mnt/mappanév - Partíció csatolása
  4. sudo nano /etc/fstab - Csatolási fájl szerkesztése
  5. /dev/meghajtó /mnt/mappa ext4 defaults 0 2 - Automatikus csatolás
  6. sudo umount /mnt/mappanév - Mindkét létrehozott partíció csatolása
  7. sudo apt update && sudo apt install mdadm -y - Szükséges RAID csomagok telepítése
  8. sudo mdadm --create /dev/md/RAID mappa név -l 1 -n 2 /dev/1.meghajtó1 /dev/2.meghajtó1 - RAID 1 tükrözése lérehozás a 2 lemezre
  9. sudo mkfs.ext4 /dev/md/RAID mappa név - RAID formázása
  10. sudo mount /dev/md/RAID mappa név - RAID csatolása

sudo systemctl status mdmonitor - RAID állapotának ellenőrzése

cat /proc/mdstat - RAID állapotának ellenőrzése
