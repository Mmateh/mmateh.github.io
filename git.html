Linux parancsok

Fájlok

ls - Listázás

nano fájlnév - Szövegszerkesztő megnyitása a megadott fájlnévvel

cat fájlnév - A megadott fájl tartalmának kiírása a képernyőre

Mappák

mkdir mappanév - Új mappa létrehozása a jelenlegi könyvtárban

cd mappanév - Belépés a megadott nevű mappába

cd .. - Visszalépés egy szinttel a mappa hierarchiában

rmdir mappanév - A megadott mappa törlése (csak akkor, ha üres)

pwd - A jelenlegi munkakönyvtár teljes elérési útvonala

tree - A jelenlegi könyvtárban található mappák és fájlok hierarchiáját ábrázoló fa-struktúra kiírása

Egyéb

echo - A megadott szöveg kiírása a képernyőre

Windows parancsok

Alapok

parancs /? - Kiírja a parancs lehetőségeit

parancs | findstr szöveg - Kiírja a megadott szöveget tartalmazó eredményét az adott parancsnak

parancs | clip - Kiírja a parancs kimenetelét a vágó memóriába

help - Kiírja az összes parancsot

cls - Parancssor törlése a konzolból

ping IPcím vagy webcím - 4 jelet küld és kiírja a választ

ping -t IPcím vagy webcím - Folyamatos jelet küld és kiírja a választ

Lemezkezelés

chkdsk - Kiírja a lemez információit

chkdsk /f - A megadott lemezen ellenőrzi a fileokat és javítja a hibákat

chkdsk /r - A megadott lemezen ellenőrzi a szektorokat és javítja a hibákat

sfc /scannow - Ellenőrzi a rendszerfileokat és javítja a hibákat

format - Leformáz egy meghajtót

DISM /Online /Cleanup-Image /CheckHealth - Gyors ellenőrzést végez

DISM /Online /Cleanup-Image /ScanHealth - Alapos ellenőrzést végez

DISM /Online /Cleanup-Image /RestoreHealth - Helyreállítja a problémákat

Filekezelés

meghajtó: - Az adott meghajtóba lép

dir - Kiírja a mappa tartalmát

cd mappanév - A megadott mappába lép

cd .. - Kilép a mappából

mkdir név - Létrehoz egy mappát a megadott néven

rmdir név - Kitörli a megadott nevű mappát

type filenév - Kiírja a file tartalmát a terminálba

assoc - Megadja, hogy az adott filetípust milyen alkalmazással nyitja meg

assoc .filetípus=program - Az adott filetípusnak beállítja a megnyitáshoz használandó alkalmazást

tasklist - Kiírja a futó fájlokat

taskkill /f /pid fileID - Leállítja a megadott filet

IP config

ipconfig - IP címének lekérdezése

ipconfig /release - IP cím törlése

ipconfig /renew - IP cím újratöltése

ipconfig /all - Minden IP cím lekérdezése

ipconfig /displaydns - DNS címlekérdezése

ipconfig /flushdns - DNS Resolver Cache törlése

ipconfig /release-dns - DNS cím törlése

Hálózat

nslookup webcím - A megadott webcím IP címét visszaadja

tracert IPcím vagy webcím - Megmutatja a megtett utat a megadott címhez

netstat - Megadja, hogy mire vagyunk csatlakoztatva vagy mi van ránk csatlakozva

netstat -af - Megadja a megnyitott portokat

netstat -o - Megadja a megnyitott portokat PID-val együtt

netstat -e -t szám - Portstatisztikát ad meg a megadott számú másodpercenként

route print - Kiírja az utat amit adott weboldalakhoz tenne meg a szamítógép

route add cél_IPcím mask maszk_IPcím gateway_IPcím - A megadott cél IP címnek létrehoz egy utat a megadott gatewayen keresztül

route delete IPcím - Törli a megadott cél IP címnek létrehozott utat

Egyéb

shutdown /s - Leállítja a gépet

shutdown /s -t szám - Leállítja a gépet a megadott számú másodperc után

shutdown /a - A számítógép léállításának megszakítása

shutdown /r /fw /f /t szám - A megadott számú másodperc után újraindítja a gépet és a BIOS-ba lép

netsh wlan show wlanreport - Készít egy jelentést a WLAN-ról

netsh interface show interface - Megmutatja az interfaceket

netsh interface ip show address - Az interfacek IP címének lekérdezése

netsh interface ip show dnsservers - Az interfacek DNS címének lekérdezése

netsh advfirewall set allprofiles state off - Kikapcsolja a tűzfalat

netsh advfirewall set allprofiles state on - Bekapcsolja a tűzfalat

getmac /v - A MAC-címeket írja ki

Energia

powercfg /energy - Az energiahasználattal kapcsolatban ad meg információt

powercfg /batteryreport - Az akkumlátorral kapcsolatban ad meg információt

Git parancsok

Github

git config --global user.name felhasználónév - Regisztrálja a felhasználónevet

git config --global user.email email_cím - Regisztrálja az emailt

git clone github_link - Leklónozza a github mappát és tartalmát

Fájlok és mappák

git init mappanév - Létrehozza a gitet

git add fájlnév - Hozzáadja a megadott fájlt a git előkészítő részére

git add * - Hozzáad minden fájlt a git előkészítő részére

git commit -m “Üzenet” - Feltölti a fájlokat egy üzenettel az előkészítőből a gitre

git log - Listázza a verziókat

Elágazások

git branch - Listázza az elágazásokat

git branch elágazásnév - Létrehoz egy elágazást

git branch -d elágazásnév - Törli az elágazást

git checkout elágazásnév - Belép az elágazásba

git diff - Megadja a különbséget két verzió között
